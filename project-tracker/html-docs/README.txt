HOW TO CONVERT TO PDF:
----------------------

Method 1: Using Your Browser (Recommended)
1. Open "index.html" in any web browser (Chrome, Firefox, Safari, Edge)
2. Click on any document you want to convert
3. Press Ctrl+P (Windows/Linux) or Cmd+P (Mac)
4. Select "Save as PDF" as the destination
5. Click Save

This will give you a properly formatted PDF with:
- Professional styling
- Syntax-highlighted code blocks
- Formatted tables
- Working internal links

Method 2: Batch Convert (If you have Node.js)
Run this command in the project-tracker folder:
